# 🌐 GitHub Pages 배포 가이드

## ✅ 자동 배포 설정 완료!

GitHub Actions workflow가 설정되어 있어서 코드를 푸시하면 **자동으로 배포**됩니다!

---

## 🚀 배포 활성화하기 (1분 완성!)

### 1️⃣ GitHub 저장소 설정

1. **GitHub 저장소로 이동**
   ```
   https://github.com/parkh37t/project_manage
   ```

2. **Settings 탭 클릭**

3. **왼쪽 메뉴에서 "Pages" 클릭**

4. **Source 설정**
   - Source: `GitHub Actions` 선택

5. **완료!** 🎉

---

## 🌍 배포된 URL

설정 후 5분 정도 기다리면 다음 URL에서 접속 가능합니다:

```
https://parkh37t.github.io/project_manage/
```

---

## 🔄 자동 배포 프로세스

코드를 푸시하면:
1. ✅ GitHub Actions가 자동 실행
2. ✅ 프로젝트 빌드
3. ✅ GitHub Pages에 배포
4. ✅ 5분 후 업데이트 완료

---

## 📊 배포 상태 확인

### GitHub Actions 확인
```
https://github.com/parkh37t/project_manage/actions
```

- ✅ 초록색 체크: 배포 성공
- ⏳ 노란색 원: 배포 진행 중
- ❌ 빨간색 X: 배포 실패
